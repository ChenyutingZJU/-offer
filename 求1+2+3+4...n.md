# 求1+2+3+4...n
### 题目描述
求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

### 解题思路
* 使用逻辑与`&&`计算的特点，即`a&&b`，如果前面为假，则后面的`b`就不会再判断了

```
class Solution {
public:
    int Sum_Solution(int n) {
        int sum =n;
        (n>0) && (sum = sum + Sum_Solution(n-1));
        return sum;
    }
};
```
